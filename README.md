## 🔍 SEO Setup

* Sitemap: `/sitemap.xml`
* Robots: `/robots.txt`
* Sudah didaftarkan ke Google Search Console
* Struktur URL SEO-friendly (tanpa parameter aneh)

---

## ⚙️ Setup & Deploy

### 1. Clone repo

```
git clone https://github.com/Avenixa/AVENIXA.git
```

### 2. Edit file sesuai kebutuhan

* Update konten HTML
* Tambah artikel di folder `pages/`

### 3. Upload ke hosting

* Upload semua file ke `public_html/`
* Pastikan `sitemap.xml` dan `robots.txt` ada di root

---

## ⚠️ Catatan penting

* Ini web statis, tidak ada backend
* Jangan taruh logic server di sini, karena memang tidak ada servernya
* Kalau ada bug SEO, biasanya bukan karena XML, tapi karena hosting atau routing

---

## 🧠 Known Issues

* Google Search Console kadang lambat update sitemap
* Hosting bisa override routing kalau konfigurasi salah
* SPA fallback bisa ganggu file seperti sitemap.xml
